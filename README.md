Serverless technical SEO auditor for Next.js/Vercel sites. Crawls sitemaps, runs Lighthouse, validates schema/hreflang, AI insights on E-E-A-T/CWV.

## Features
- Bulk Lighthouse (perf/SEO/access).
- Crawl orphans/redirects.
- Schema, robots.txt, mobile checks.
- Prioritized fixes w/ impact est.
- Clerk auth, Vercel Postgres, Tremor charts.

## Quickstart
pnpm i
pnpm dev

## Deploy
Connect GitHub to Vercel, add env vars (Clerk keys, DATABASE_URL).

See docs/DEPLOY.md
